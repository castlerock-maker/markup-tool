Markup Tool

Annotate any webpage. Export feedback as JSON for Claude Code.

1

Install the bookmarklet

Drag this button to your bookmarks bar:

Markup Tool

Can't see the bookmarks bar? Press Cmd+Shift+B (Mac) or Ctrl+Shift+B (Windows)

2

Use it on any page

Navigate to any webpage (including localhost), then click the bookmarklet in your bookmarks bar. A sidebar will appear on the right side of the page.

3

Highlight & annotate

Select any text on the page. A popup will appear where you can type your feedback comment — what needs to change about the highlighted content. Press Enter to save.

4

Export for Claude Code

Click Export JSON in the sidebar footer. The JSON is copied to your clipboard — paste it directly into a Claude Code conversation to request changes.

↓

What's in the export

Selected text The exact content you highlighted
Your comment What change is needed
DOM context Element path + nearest heading
Page info URL, title, timestamp

Annotations are saved in localStorage per page URL and persist across sessions.
Click the bookmarklet again to toggle the sidebar.
